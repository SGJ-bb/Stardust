> 版本: v0-douyin-tech | 初始化日期: 2026-05-22
> 基于: opinion-video-zero 等权占位 + 抖音技能展示类视频定制

## 7 维度评分体系

每维度 1-10 分。composite = 加权平均。

| # | 维度 | 代码 | 权重 | 核心问题 |
|---|------|------|------|----------|
| 1 | Hook 力度 | HK | 0.20 | 前3秒是否让人停下来？ |
| 2 | 信息差冲击 | IG | 0.15 | 观众看完是否获得了"原来如此"的新知？ |
| 3 | 情绪共鸣 | ER | 0.15 | 是否触发了"卧槽""我也想学""太强了"等情绪？ |
| 4 | 展示可信度 | SC | 0.15 | 实际展示的技能/项目是否真实、可验证？ |
| 5 | 完播设计 | RD | 0.15 | 是否有让人看完的结构（悬念/递进/反转）？ |
| 6 | 人设辨识度 | PD | 0.10 | "大一学生一个月学会编程"标签是否突出？ |
| 7 | 互动钩子 | IH | 0.10 | 是否引导了评论/点赞/关注行为？ |

**Composite 公式**: `HK×0.20 + IG×0.15 + ER×0.15 + SC×0.15 + RD×0.15 + PD×0.10 + IH×0.10`

---

## 各维度详细锚点

### 1. Hook 力度 (HK) — 权重 0.20

抖音前3秒决定生死。Hook 不行，后面再好也没人看。

| 分数 | 描述 |
|------|------|
| 1-2 | 开头无冲击力，"大家好我是XX"式开场 |
| 3-4 | 有尝试但不够狠，信息量不足 |
| 5-6 | 有明确钩子但不够意外 |
| 7-8 | 前3秒制造认知冲突或强烈好奇 |
| 9-10 | 前3秒让人"必须看完"，产生生理级注意力锁定 |

**高分锚点**: "一个月前我还不会写代码，现在我自己做了个AI女朋友" → 认知冲突+身份反差

### 2. 信息差冲击 (IG) — 权重 0.15

观众是否获得了"原来还能这样"的新知。

| 分数 | 描述 |
|------|------|
| 1-2 | 观众早已知道的信息 |
| 3-4 | 有一点新意但不够震撼 |
| 5-6 | 提供了明确的新信息/新视角 |
| 7-8 | 打破常识，"原来可以这样" |
| 9-10 | 降维打击级信息差，看完世界观被刷新 |

**高分锚点**: 展示AI伴侣App的某个功能，观众没想到个人开发者能做到这种程度

### 3. 情绪共鸣 (ER) — 权重 0.15

是否触发了强烈的情绪反应。

| 分数 | 描述 |
|------|------|
| 1-2 | 平铺直叙，无情绪波动 |
| 3-4 | 有一点感觉但不够强烈 |
| 5-6 | 有明确的情绪点 |
| 7-8 | 引发"卧槽""我也想""太牛了" |
| 9-10 | 情绪爆发，忍不住分享/评论 |

**高分锚点**: 展示从零到成品的对比，触发"我也能做到"的渴望

### 4. 展示可信度 (SC) — 权重 0.15

实际展示是否真实、可验证。抖音观众对"假的"极度敏感。

| 分数 | 描述 |
|------|------|
| 1-2 | 只有口述没有展示 |
| 3-4 | 有展示但不够具体 |
| 5-6 | 有实际画面/操作展示 |
| 7-8 | 真实操作+可验证细节 |
| 9-10 | 完整演示+源码/下载链接等可验证 |

**高分锚点**: 实际操作App界面，展示代码编辑器，真机运行画面

### 5. 完播设计 (RD) — 权重 0.15

是否有让人看完的结构设计。

| 分数 | 描述 |
|------|------|
| 1-2 | 无结构，想到哪说到哪 |
| 3-4 | 有基本结构但缺乏悬念 |
| 5-6 | 有开头中间结尾的完整结构 |
| 7-8 | 有悬念/递进/反转，看完才满足 |
| 9-10 | 每一秒都有"不看完会亏"的感觉 |

**高分锚点**: "先看效果→你猜我花了多久→一个月前我还不会写代码→这是怎么做到的"

### 6. 人设辨识度 (PD) — 权重 0.10

"大一学生一个月学会编程"这个标签是否突出。

| 分数 | 描述 |
|------|------|
| 1-2 | 看不出是谁在做 |
| 3-4 | 有身份信息但不突出 |
| 5-6 | 人设清晰但不够独特 |
| 7-8 | 人设独特且与内容强关联 |
| 9-10 | 人设本身成为内容的一部分，不可替代 |

**高分锚点**: "大一""一个月从零""独立开发"三个标签形成独特三角

### 7. 互动钩子 (IH) — 权重 0.10

是否引导了评论/点赞/关注行为。

| 分数 | 描述 |
|------|------|
| 1-2 | 无任何互动引导 |
| 3-4 | 有"点赞关注"但太模板化 |
| 5-6 | 有具体的互动引导 |
| 7-8 | 互动引导与内容自然融合 |
| 9-10 | 观众忍不住互动（提问/挑战/分享欲） |

**高分锚点**: "你觉得这个App值多少钱？评论区告诉我" → 具体且与内容相关

## 流量预测 bucket（抖音）

基于 composite 分数预测视频播放量区间：

| Composite | 预测播放量 | 信号 |
|-----------|-----------|------|
| 1-3 | < 200 | 基础流量池都没出 |
| 3-4 | 200-500 | 进了基础池但没破圈 |
| 4-5 | 500-1K | 破了基础池 |
| 5-6 | 1K-3K | 进了二级推荐 |
| 6-7 | 3K-1W | 稳定二级推荐 |
| 7-8 | 1W-5W | 破圈进入大流量池 |
| 8-9 | 5W-20W | 大流量池 |
| 9-10 | 20W+ | 爆款级 |

> ⚠️ 当前 bucket 是 v0 先验估计，未经校准。5+ 条复盘数据后走 bump 流程修正。
